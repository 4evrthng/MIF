Trys dalys: HTML, CSS ir JavaScript/JQuery (1 balas, atsiskaityti galima iki paskutinių pratybų)

HTML, CSS (0.5 balo):

reikia parašyti HTML puslapį, kuriame būtų naudojamos tik struktūrinės žymės (išvardintos HTML5 dokumente).

Negalima naudoti puslapio stilių nusakančių žymių bei atributų (color, font, size, ir panašiai).

Stilio aprašymui reikia sukurti stilių lentelę, aprašytą atskirame faile. Turi būti patenkinti šie reikalavimai:

Dokumentas privalo būti validus HTML5 dokumentas. Validatoriai:

  • http://validator.nu
  • http://validator.w3.org

Reikia panaudoti šiuos selektorius (ir mokėti paaiškinti, kuo pirmas skiriasi nuo antro, bei ką jie visi reiškia):

  1. Įpėdinio (descendant);
  2. Vaiko (child);
  3. Atributo (attribute);
  4. Klasės (class);
  5. ID
Sukurtoje stilių lentelėje turi būti savybių paveldėjimo pavyzdys, t.y., būtina prasmingai panaudoti raktinį žodį inherit (t.y., ištrynus šį raktinį žodį, vaizdas ekrane turi pasikeisti) ir žinoti, ką jis reiškia;

Sukurtoje stilių lentelėje turi būti kaskadų mechanizmo pavyzdys, būtina prasmingai panaudoti raktinį žodį !important (t.y., ištrynus šį raktinį žodį, vaizdas ekrane turi pasikeisti), bei žinoti, ką jis reiškia ir kas yra tas kaskadų mechanizmas;
